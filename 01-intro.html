Coding and creating a website.

Step 1

The first step I had to do was to download two applications Git Bash and Notebook this was due to the computer I use for school being a windows. Downloading these two apps will allow me to write in a terminal just like the one found on mac which allows for an easier way to teach everyone. From here I created an account on Github and followed the instructional video to make a repository. This repository will host my website and allow whoever is reading this to actually see what I've written.

After creating the repository I had to find a template I could use for my website on a site called w3schools. It took a bit of time to find something that would fit my style but I ended up using the "Band Template"

The template allows for easy access to the source code which I can later use within another program I downloaded called brackets with an extention to help push code to the reposititory. Brackets allows the user to code their website with realitive ease to an experienced creator, unfortunatley I didn't have that luxary so I used the tutorial given to test out the program it was just a simple "hello world".

After getting a slight handle on the program I copy and paste the code from my website template to mess around and see what will suit my own website. While this happened I ran into several issues due to my lack of skill with the program causing many errors and head aches. These issues while they made me want to scream has taught me so much about brackets for better or for worse. Eventually however I fixed things enough to be able to produce this webpage. ,
