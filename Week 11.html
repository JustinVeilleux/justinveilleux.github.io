The first video shown to us gives a brief explanation waling us through how to use tinkercads circuits. The first thing we did was set up an LED light with a resistor to prevent the bulb from exploding. Afterward, we added an LED and button to teach us how to turn on the light by pushing the button. Finally, to finish off the introduction video we learned how to use a potentiometer to be able to adjust the resistance and in turn light levels of an LED.

The second video gave us a brief introduction to coding in Arduino. I have had a slight bit of coding experience so this style wasn't too difficult for me to understand. From the start, we learned a code that could do basic calculations based on money. through the serial monitor we would be able to see how our code was working with this in mind we were able to use the basic calculating skills to show us the remaining value of our money with any of the functions we wanted to use. + would add, - would subtract, * would multiply, and % would divide. After making the basic calculator we learned how to add on phrases like "I'm rich!" or "I'm not rich" based on the values we get from plugging our equations in the code.

In the third video, we learned how to connect both code and building in tinkercad. The additional code learned is meant to help us learn how to work with the LED lights for our assigned work. The code allowed us to blink our LED light by pushing the button.
